# Cloud Utils

## What?

A collection of misc utils for working with AWS resources.

## Usage

```bash
cloud-utils
```

### Install/Upgrade

```bash
npm i -g 'git@github.com:onebytegone/cloud-utils.git'
```

### Install Bash Autocompletion

```bash
cloud-utils --install-autocompletion
```
